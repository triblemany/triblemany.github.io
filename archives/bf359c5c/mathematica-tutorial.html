Mathematica的常用操作和常用函数

Posted on 2019-11-21 Edited on 2020-01-31
Symbols count in article: 2.2k

本文介绍了Mathematica的常用操作和常用函数。

常用命令

  • 运行代码 Shift+Enter

  • 中止运行 Alt+.

  • 输入下标 Ctrl+_

  • 输入上标 Ctrl+^

  • 插入根式 Ctrl+2

  • 插入分数 Ctrl+/

  • 输入特殊字符 Esc

常用语法

  • list = {1, 2, 3} ( 创建列表 )
  • list[[1]] ( 访问列表元素，从1开始 )
  • rule = a->b ( 创建规则，用a替换b )
  • 3 b /. rule ( 应用规则（替换） )
  • f@x ( 将函数f应用于x，即f[x] )
  • x // f ( 将函数f应用于x，即f[x] )
  • add[x_, y_]:=(x=x+2, y=y+2, x+y) ( 定义函数，下划线表示接受任意类型，输出最后一个值 )

常用常量

  • I, 单位虚数
  • Pi, 圆周率
  • E, 自然常数

常用函数

  • Solve[a+b==2 && a-b==0, {a, b}] ( 解方程 )
  • Collect[(x+1)^2， x] ( 合并同类项 )
  • Expand[(x+1)^2] ( 展开 )
  • ComplexExpand[(x+I)^2] ( 复数展开 )
  • Simplify[x^2+2 x+1] ( 化简 )
  • Numerator[a/b] ( 获得分子 )
  • Denominator[a/b] ( 获得分母 )
  • D[x^n, x] ( 求导数 )
  • N[Pi] ( 数值计算 )
  • Re[1+2 I] ( 取实部 )
  • Im[1+2 I] ( 取虚部 )
  • Integrate[f, x] ( 求不定积分 )
  • Integrate[f, { x, xmin, xmax } ] ( 求不定积分 )
  • Plot[{f, g}, { x, xmin, xmax } ] ( 绘制图像 )
  • Module[{x, y}, x=2; y=3; x+y] ( 定义和使用局部变量 )
  • Clear[“Global`“*] 清除所有变量

矩阵计算函数

  • Det[{ {1, 0}, {0, 1} }] ( 求矩阵的行列式 )
  • Inverse[{ {1, 0}, {0, 1} }] ( 求矩阵的逆 )
  • Transpose[{ {1, 0}, {0, 1} }] ( 求矩阵的转置 )
  • ConjugateTranspose[{ {1, 0}, {0, 1} }] ( 求矩阵的共轭转置 )
  • MatrixForm[{ {1, 0}, {0, 1} }] ( 以矩阵形式输出 )
  • IdentityMatrix[3] ( 给出单位矩阵 )
  • Dot[a, b, c] ( 矩阵点积，或用. )
